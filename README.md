# Find and Replace Tool (Angular)

## Features
- Text area for users to input or paste text.
- Input fields for specifying the word/phrase to find and replace.
- Button to perform the replacement operation.
- Reset button to clear the text area and inputs.

## How to Use
1. Type or paste your text in the text area.
2. Enter the word or phrase to find.
3. Enter the word or phrase to replace with.
4. Click the "Replace" button to perform the operation.

## Assumptions
- The replacement is case-sensitive.
- The `find` input cannot be empty.
- Replacement happens for all occurrences of the `find` word/phrase.

## Tech Stack
- Angular 17
- Bootstrap (optional for styling)

## To Run Locally
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the development server: `ng serve`.
4. Open your browser at `http://localhost:4200`.
